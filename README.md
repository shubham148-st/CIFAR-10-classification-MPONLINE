# CIFAR-10-classification-MPONLINE

Dataset Used:  CIFAR-10 dataset.
This dataset contains 60,000 32x32 color images in 10 classes (Airplane, Automobile, Bird, Cat, Deer, Dog, Frog, Horse, Ship, Truck).

total epochs: 20,
lr: 0.001
